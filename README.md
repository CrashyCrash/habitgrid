# HabitGrid — Visual Habit Tracker

[![Live Demo](https://img.shields.io/badge/demo-live-22c55e?style=for-the-badge)](https://crashycrash.github.io/habitgrid/)
[![Built with React](https://img.shields.io/badge/React-19-61dafb?style=flat-square&logo=react)](https://react.dev)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.8-3178c6?style=flat-square&logo=typescript)](https://typescriptlang.org)

Track your daily habits with beautiful **GitHub-style contribution grids**. 
Your data stays on your device — private, fast, and works offline.

![HabitGrid Preview](https://img.shields.io/badge/status-live-22c55e)

## Features

- **GitHub-style contribution grids** — Beautiful heatmap visualization for every habit
- **Streak tracking** — Current streak, longest streak, and completion rates
- **Multiple habits** — Track unlimited habits with custom colors and icons
- **Privacy first** — All data stored in localStorage, never leaves your device
- **Works offline** — PWA-ready, install it on your phone
- **Dark theme** — Beautiful glassmorphism design
- **Import/Export** — Backup and restore your data as JSON
- **responsive** — Works great on mobile, tablet, and desktop

## Tech Stack

- React 19 + TypeScript 5.8
- Vite 6 (build in <1s)
- Tailwind CSS 4
- Lucide React icons
- Zero backend, zero tracking, zero cookies

## Getting Started

```bash
npm install
npm run dev     # Development server
npm run build   # Production build
```
